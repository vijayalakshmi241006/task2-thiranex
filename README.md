# Customer Segmentation Analysis using K-Means Clustering

## Project Overview
This project focuses on segmenting customers based on their purchasing behavior and demographic characteristics using Machine Learning clustering techniques.

The goal is to identify different customer groups and generate insights that can help businesses understand customer preferences and create targeted marketing strategies.

## Objective
- Perform customer segmentation using K-Means clustering
- Analyze customer behavior and purchase patterns
- Identify different customer groups based on spending habits
- Visualize customer segments and extract business insights

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- K-Means Clustering
- Data Visualization

## Dataset Description
A customer dataset was created containing customer information such as:

- Customer ID
- Age
- Gender
- Region
- Annual Income
- Spending Score
- Purchase Frequency
- Average Order Value

The dataset contains customer behavior and demographic attributes used for segmentation analysis.

## Methodology

### 1. Data Preparation
- Generated and explored customer data
- Checked dataset structure and statistics
- Selected relevant features for clustering

### 2. Data Preprocessing
- Applied feature scaling using StandardScaler
- Prepared data for machine learning model

### 3. Customer Segmentation
- Applied K-Means clustering algorithm
- Used Elbow Method and Silhouette Score to determine the optimal number of clusters

### 4. Cluster Analysis
Customers were grouped into different segments based on their characteristics:

- Budget Shoppers
- High-Value Loyals
- Young Explorers
- Occasional Buyers
- Premium Segment

### 5. Visualization
Created visualizations to understand customer segments:

- Elbow Method & Silhouette Score
- Income vs Spending Score Scatter Plot
- Customer Segment Size Distribution
- Average Order Value by Segment

## Key Insights
- Different customer groups were identified based on purchasing patterns
- High-value customers can be targeted with loyalty programs
- Budget customers can be engaged through offers and discounts
- Premium customers can receive personalized recommendations
- Customer segmentation helps businesses improve marketing strategies
